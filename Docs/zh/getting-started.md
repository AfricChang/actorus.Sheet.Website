## AcTorus.Sheet 适合做什么

AcTorus.Sheet 是一个适合 `.NET` 团队使用的电子表格引擎，重点面向 `.xlsx` 文件处理和自动化工作流。

它适合做这些事：

- 打开工作簿文件
- 读取和更新工作簿数据
- 计算公式
- 保存处理后的工作簿结果

## 建议的开始顺序

如果你第一次接触产品，可以按这个顺序：

1. 先看功能概览。
2. 再看限制说明。
3. 打开下载页。
4. 试着跑通第一个工作簿流程。

## 第一个工作流通常是什么样

一个典型的入门流程通常是：

1. 打开工作簿，或者创建一个新工作簿。
2. 写入单元格数据。
3. 计算公式。
4. 保存为 `.xlsx`。
