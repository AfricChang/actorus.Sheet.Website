## 文件范围

当前公开产品重点支持 `.xlsx`。

## 产品范围

AcTorus.Sheet 的定位是自动化和系统集成，不是桌面电子表格编辑器。

## 当前不包含的内容

当前公开范围不包括：

- 桌面电子表格 UI
- 工作簿设计器
- 宏或 VBA 执行
- 浏览器电子表格控件
- 将非 `.xlsx` 格式作为当前公开承诺

## 功能预期

产品覆盖的是“实用而明确”的工作簿和计算能力，但并不等于完整桌面电子表格软件的全部功能。

如果你的业务流程依赖高级或特殊行为，建议在正式采用前先验证适配性。
