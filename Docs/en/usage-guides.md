## Report generation

Use workbook templates, fill in values, calculate formulas, and save finished files for download or delivery.

## Batch workbook processing

Use the engine in scheduled jobs or internal processing pipelines when many workbooks need the same repeated handling.

## Backend calculation

Put spreadsheet logic into your backend workflows when workbook formulas are part of validation, result generation, or business processing.

## Product integration

Embed the engine in your own product when spreadsheet support needs to become part of your existing application instead of a separate manual task.

## A good evaluation path

If you are not sure whether your workflow fits:

1. Start with a small representative workbook.
2. Confirm the needed file features.
3. Confirm the needed formula families.
4. Use the contact page if you need trial access or a deeper discussion.
