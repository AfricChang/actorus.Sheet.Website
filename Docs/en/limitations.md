## File scope

The current public product focus is `.xlsx`.

## Product scope

AcTorus.Sheet is designed for automation and integration. It is not a desktop spreadsheet editor.

## Not included

The current public scope does not include:

- desktop spreadsheet UI
- workbook designers
- macros or VBA execution
- browser spreadsheet controls
- non-`.xlsx` file formats as a core public promise

## Feature expectations

The product supports a practical subset of workbook and calculation features, but not every feature found in full desktop spreadsheet software.

If your workflow depends on advanced or specialized workbook behavior, validate the fit before adoption.
