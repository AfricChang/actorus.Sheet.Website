## What AcTorus.Sheet is for

AcTorus.Sheet is a spreadsheet engine for `.NET` teams that need to work with `.xlsx` files in applications and backend workflows.

It is built for tasks such as:

- opening workbook files
- reading and updating workbook data
- calculating formulas
- saving processed workbook results

## A simple way to start

Use this order when you are new to the product:

1. Read the feature overview.
2. Check the limitations page.
3. Open the public download page.
4. Try your first workbook flow.

## Your first workflow

A typical first workflow looks like this:

1. Open a workbook or create a new one.
2. Write values into cells.
3. Calculate formulas.
4. Save the workbook to `.xlsx`.
