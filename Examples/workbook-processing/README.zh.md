# 工作簿处理示例

演示完整的端到端工作簿流程：创建、写入数据、计算公式、保存为 `.xlsx`，再重新打开并验证结果。

## 涵盖内容

- 创建工作簿并写入单元格值和公式
- 通过 `Calculate()` 执行公式计算
- 设置冻结窗格和数据验证
- 通过 `Save()` 将工作簿保存为字节数组
- 通过 `OpenWorkbook()` 重新打开已保存的工作簿并读取结果

## 运行方式

```bash
dotnet run
```

无外部依赖。示例完全在内存中运行，结果输出到控制台。

## 适合的场景

报表生成、模板填充和自动化工作簿导出——这通常是接入引擎时最先验证的一条流程。
